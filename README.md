사용할 기술적인 사항

UI에 쓰일 ImageSwitcher, Custom Drawing에서 커스텀 layout과
저번 수업에서 배운 image들 움직이는것
+CollisionBox구성과 총알과의 충돌

 

배워야할 사항

양손조작(조이스틱과 비슷하여 화면 아래 양 끝에 배치)
좌측은 상하좌우 움직임
우측은 회전
spriteAnimation(image animation)
+@
